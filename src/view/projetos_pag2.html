Análise de Dados de Comércio Exterior no estado de São Paulo

Seguindo a metodologia SCRUM, o objetivo deste projeto é desenvolver uma plataforma web que disponibilize informações sobre o desempenho dos municípios do Estado de São Paulo no comércio exterior, com base nos dados abertos do Ministério do Desenvolvimento, Indústria, Comércio e Serviços. Essa ferramenta fornecerá aos tomadores de decisão dados claros e acessíveis, permitindo a identificação de municípios que estejam em ascensão, estagnação ou declínio no mercado internacional.

Como parte da equipe de Desenvolvimento minha tarefa é normalizar a base de dados utilizados para análise, criar a lista de dados de exportação de todos os municípios de São Paulo e criar uma ferramenta de busca para filtrar cargas por código NCM com aplicação de filtros.
